================================================================================
  SECTOR 3 — MULTIMODAL LATE FUSION SIMULATION
  ASD Detection: Facial Stream + Q-CHAT-10 Stream
================================================================================

PURPOSE
-------
Feasibility analysis of weighted late-fusion combining:
  - Facial stream  : VGG-Face CNN (sector1/Stage_4), 256-D embedding + classifier
  - Q-CHAT stream  : XGBoost parent-only questionnaire (sector2/Stage_2)

Fusion rule: P_final = alpha x P_facial + (1 - alpha) x P_qchat
Method     : Monte Carlo simulation — 500 iterations x 300 stratified pairs per
             alpha value. Pairs are class-matched (ASD-ASD, non-non) since the
             two datasets have NO patient overlap.

METHODOLOGICAL NOTE
-------------------
The facial and Q-CHAT datasets are entirely independent populations. This
simulation estimates population-level behaviour of a hypothetical dual-modal
system by Monte Carlo resampling. Results are NOT individual-level validation.
A prospective study with the same children assessed by both modalities is
required for definitive paired validation.

  - Adding too much facial weight (alpha > 0.5) substantially degrades all
    metrics because the facial model (AUC=0.858) introduces noise.

  SECTION C — FAILURE MODE COMPLEMENTARITY
================================================================================

Analysis at alpha=0.15, 500 MC iterations x 150 ASD pairs each.
Focuses on ASD-positive cases only.

RESULTS
-------
  Facial standalone ASD recall           : 80.9%
  Q-CHAT standalone ASD recall           : 98.8%

  When facial misses ASD (FN)
  -> Q-CHAT catches                       : 98.8% +/- 2.2%

  When Q-CHAT misses ASD (FN)
  -> Facial catches                       : 81.5% +/- 30.2%

  Both miss (irreducible joint FN rate)   : 0.23% +/- 0.40%

PLOT: complementarity_analysis.png
  - Q-CHAT standalone recall (98.8%) is 18 percentage points above facial (80.9%)
  - When facial generates a FN, Q-CHAT catches 98.8% of those cases — the
    Q-CHAT model is an almost complete safety net for facial failures
  - When Q-CHAT generates a FN, facial catches 81.5% — but the very wide
    error bar (+/-30.2%) reflects that Q-CHAT FN cases are rare, making
    this estimate noisy across MC iterations
  - The combined irreducible FN rate is just 0.23% — nearly all ASD cases
    are caught by at least one stream; almost no individual slips through both

INTERPRETATION:
  - The primary clinical value of fusion is NOT the +0.0003 AUC gain but the
    near-elimination of false negatives: 0.23% joint FN vs 19.1% facial alone.
  - Q-CHAT is the dominant safety net; facial serves as a backup for the rare
    Q-CHAT failures, recovering ~81.5% of them.

================================================================================
  SECTION F — CLINICAL INTERPRETATION
================================================================================

1. Q-CHAT DOMINANCE
   The parent-completed Q-CHAT-10 (sector2/Stage_2, parent-only population)
   reaches near-perfect AUC=0.9991. This is the result of:
     (a) Filtering to parent-completed responses only (sector2 Stage_2 finding)
     (b) Q-CHAT-10 being specifically designed as an ASD screening instrument
   The facial model (AUC=0.8578) is substantially weaker in this comparison.

2. FUSION VALUE IS NOT AUC IMPROVEMENT
   The AUC gain from fusion is only +0.0003 — statistically insignificant given
   that Q-CHAT already operates near the theoretical ceiling. Fusion's value lies
   entirely in complementarity: the joint irreducible FN rate drops to 0.23%,
   versus 19.1% for facial alone and 1.2% for Q-CHAT alone.

3. COMPLEMENTARITY IS THE CLINICAL ARGUMENT
   - Q-CHAT can fail when parents lack accurate behavioural recall or
     misinterpret questions (Q-CHAT FN rate ~1.2%)
   - Facial analysis is objective, parent-independent, and non-verbal
   - When Q-CHAT fails, the facial stream recovers ~81.5% of those cases
   - The combined system's 0.23% FN rate represents a strong safety net
     for a high-stakes screening context

4. DEPLOYMENT RECOMMENDATION
   - Primary screening instrument: Q-CHAT-10 (parent questionnaire)
   - Alpha = 0.15 (15% facial weight) — use facial as a correction signal
   - Decision threshold:
       tau = 0.35 for general population screening (F2-optimal)
       tau = 0.90 for specialist referral queue (near-zero false positives)
   - The wide "safe operating window" (tau=0.40 to 0.65) gives clinicians
     flexibility to set tau based on local resource constraints

5. LOGREG PROBE INSIGHT
   LogReg probe AUC (0.8593) = VGG-Face CNN AUC (0.8578). The 256-D VGG-Face
   embeddings contain essentially all discriminative information the CNN
   captures. A future improvement path: fine-tune the embedding space for
   ASD-specific features rather than relying on general-purpose face features.
